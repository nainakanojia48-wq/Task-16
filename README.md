# Student Records

## Description
This project displays student records using HTML, CSS, and JavaScript. Student data is stored in an array of objects and displayed dynamically in card format.

## Features
- Student records displayed as cards
- Each card includes name, marks, class, and address
- Search box filters students by name
- Real-time search while typing
- Uses JavaScript map() to render student cards
- Uses JavaScript filter() to search student records

## Technologies Used
- HTML
- CSS
- JavaScript

## Files
- index.html
- style.css
- script.js
- README.md

## How to Run
1. Open the project folder.
2. Make sure all files are in the same folder.
3. Open index.html in the browser.
4. Student cards will appear automatically.
5. Type a name in the search box to filter records.
